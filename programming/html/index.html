HTML is a markup language.
Markup languages are not used to write programs, instead, they are used to tell programs what to do.
In fact, markup is in HTML's name!
Hyper Text Markup Language
Markup languages tell programs what to do using markup tags. Each tag is a single instruction or piece of data.
In the case of HTML, these tags are called, you guessed it, HTML tags!

Sidenote, I will often use the words "tag" and "element" interchangebly. Technically, they are not the same thing, but we will get to that later.

These HTML tags give your web browser data, which your browser uses to construct a visual webpage.
Markup languages are programming languages like any other however, and so they have their file extensions and such. For HTML, you make .html files, and these are called HTML Documents.
Here is an example of a simple HTML page.
<!DOCTYPE html>

<html>

<head>

  <title>Page Title</title>

</head>



<body>

  <h1>My First Heading</h1>

  <p>My first paragraph.</p>

</body>



</html>

My First Heading

My first paragraph.


Whoa, hold your horses, where did all that code go? What did it do? All I see is that heading and paragraph text!! What is going on!?!?!?!?
Fear not, that other code will be explained!

    Explaination
  • <!DOCTYPE html> tells the browser that this is an html file. (Why not just check if it is a .html file? Reasons, reasons that you don't need to know yet.)
  • <html> and </html> are the beginning and ending of the html document.
  • <head> and </head> contain extra information that your browser recieves, but isn't part of the webpage, for example...
  • ...<title>Page Title</title> tells the browser the title of the webpage. This is what gets shown on the tab for the webpage.
  • <body> and </body> are the beginning and ending of the body of the webpage. This is what you actually see.
  • <h1>My First Heading</h1> is the beginning, content, and ending of a heading-styled text element.
  • <p>My first paragraph.</p> is the beginning, content, and ending of a paragraph-styled text element.

So there you go, pretty simple, but at the same time pretty cool!
Now, how about these HTML tags. These things look something like this:

<tagname attribute=value attribute=value>some text or something</tagname>
(Colors provided for easier readability.)

If you know basic XML, than this might look familier to you.
HTML tags useually come in pairs. The first of the pair is a opening tag and the second is a closing tag. An end tag is exactly like its corrisponding start tag, except for that there is a slash before the tag name. Look again.
<tagname> </tagname>

HTML documents are what get sent to your browser when you access a webpage.
It is important to note that an HTML Document is not the same as an HTML File. HTML Files are any file with the .html extension, while HTML Documents are actual webpages.
<!DOCTYPE html>
 This defines that the file, regardless of the file extension, is an html file.
Within a file defined as an html document, you have to define the beginning and end of the html. You do this with the html element.
<!DOCTYPE html>

<html>



</html>
 Now that the file is an html document, and the html document start and end have been defined, we now need to define the beginning and end of what is actually displayed on the webpage. We do this with the body element.
<!DOCTYPE html>

<html>

    <body>



    </body>

</html>
 Now that we have everything that we need, now let's add in some content. How about some headings?
<!DOCTYPE html>

<html>

    <body>

        <h1>This is the biggest heading, h1</h1>

        <h6>This is the smallest heading, h6</h6>

    </body>

</html>
 If we just want some normal text, we can use the p, for paragraph, tag.
<!DOCTYPE html>

<html>

    <body>

        <h1>This is the biggest heading, h1</h1>

        <h6>This is the smallest heading, h6</h6>

        <p>Hi there! I am a paragraph text! :D</p>

    </body>

</html>
 How about another very common element, a link.
To make a link, you need to use an "a" element. Then, you have to add an href attribute for the link.
<!DOCTYPE html>

<html>

    <body>

        <h1>This is the biggest heading, h1</h1>

        <h6>This is the smallest heading, h6</h6>

        <p>Hi there! I am a paragraph text! :D</p>

        <a href="http://example.com">Click here to go to example.com!</a>

    </body>

</html>
 Next up is a pretty simple element, the line break.
You need line break elements because html does not recognize line breaks in the code.
They are very simple. All that you need is a simple <br>.
<!DOCTYPE html>

<html>

    <body>

        <h1>This is the biggest heading, h1</h1>

        <h6>This is the smallest heading, h6</h6>

        <p>Hi there! I am a paragraph text! :D</p>

        <br>

        <a href="http://example.com">Click here to go to example.com!</a>

    </body>

</html>
 And, finally, the last element for now, the img element.
This one does not have a closing tag, however, it has quite a few attributes.
The first is the src attribute, which has to be a link to the image.
The next is the alt attribute, which is displayed when the image can not be loaded.
Finally, the width and height attributes, which are self-explainatory.
<!DOCTYPE html>

<html>

    <body>

        <h1>This is the biggest heading, h1</h1>

        <h6>This is the smallest heading, h6</h6>

        <p>Hi there! I am a paragraph text! :D</p>

        <br>

        <a href="http://example.com">Click here to go to example.com!</a>

        <img src="http://upload.wikimedia.org/wikipedia/commons/c/ce/Example_image.png" alt="example image" width="100" height="100">

    </body>

</html>
